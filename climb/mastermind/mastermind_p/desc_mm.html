Mastermind

You are playing mastermind on a wide board (N different letters) using wheels to choose a letter from A-Z. Given an N letter mastermind solution and an N letter attempt, what is the minimum number of changes you need to make to win? One click of the wheel equals one change so A-C is two changes, D-Z is three.

Input

First line is the solution, second line the attempt.

Output

The minimum required steps.
