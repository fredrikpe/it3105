System of linear equations

In this problem you are to solve a system of linear equations on the form Ax = b. Feel free to use the framework you have from exercise 6 and 7, although solving it by finding the inverse matrix is not recommended. There might be a better way...

Input

First line specifies dimension of A.
Next follows A, then b. F.ex
2
1 0
0 1
1
1

Output

The solution, rounded to exactly two decimal places. One number on each line.
