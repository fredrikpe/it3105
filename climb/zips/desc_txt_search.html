Text Search

Find words in a text. In this problem you are going to count the number of occurences of different words i a text file (stdin). Letter case should not be differentiated, and punctuation ignored. As an example, if one of the search words is "frank" and the text "My name is Frank, Fraaaank! #fRaNk #frankistheman" the count is 2.

Input

On the first line are the words to be searched. Next line is the beginning of the text.

Output

The number of occurrences in the same order. Ex:
2 34 5453 4
