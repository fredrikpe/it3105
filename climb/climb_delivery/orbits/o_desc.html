In this problem we want to figure out how often planets line up. For simplicity we assume that the solar system is a perfect disc, i.e., that all planets lie in the same plane. Furthermore, we approximate the orbit times of the planets as follows: earth = 365, mars = 687, and venus = 223 (we are only considering our closest neighbours). How long will take until all three planets line up (is on day 0 of their orbits)?

Input

Each case is described by a single line containing three integers 0 <= e < 365, 0 <= m < 687, and 0 <= v < 223. Indicating which days earth, mars and venus are at their respective orbits.

Output

For each case, display the smallest number of days until the three planets will both be on day 0 of their orbits (line up). Follow the format of the sample output.

Sample Input 1 Sample Output 1 
0 0 0
364 686 222

0
1
